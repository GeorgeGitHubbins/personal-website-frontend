# Personal Website Frontend

## Project Overview
This is the React frontend for `georgegittins.com`, built with Vite and TypeScript.

## Key Features
- **Continuous Evolution:** Designed to be modified autonomously by Gemini CLI.
- **Modern Tech Stack:** React, TypeScript, Vanilla CSS.
- **Seamless Deployment:** Integrated with S3 and CloudFront.

## Getting Started
1.  `npm install`
2.  `npm run dev`
3.  `npm run build` (for deployment)

## Autonomous Mode
Consult `GEMINI.md` for specific instructions on how Gemini CLI handles autonomous updates.
